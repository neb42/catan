# lib Directory

This directory contains shared library code and configuration for the web application.

## Files

- **queryClient.ts** - TanStack Query client instance with default configuration for data fetching
- **api.ts** - API base URL and helper fetch function for making requests to the backend

## Purpose

This directory houses foundational utilities that are used across the application:

- Query client configuration for caching and refetching behavior
- API utilities for consistent HTTP request handling
- Shared configuration that multiple features depend on
